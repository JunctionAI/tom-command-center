### KEY FRAMEWORKS

**AUM Growth Formula:**
```
New AUM = (Existing AUM × Organic Growth Rate) + (New Client Acquisitions × Avg Account Size) - (Client Departures × Avg Account Size)
```

**Client Lifetime Value (Wealth Management Version):**
```
LTV = (Annual Fee Rate × Avg Account Size × Average Relationship Length) - Acquisition Cost
```

**HNW Acquisition Sources (typical):**
- Referrals (60-70% of high-quality clients)
- Direct outreach / networking (15-20%)
- Content/thought leadership (10-15%)
- Strategic partnerships (5-10%)
