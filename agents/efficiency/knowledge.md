# Efficiency Auditor Knowledge — Time Economics & ROI Calculation

## ROI Frameworks

### Financial ROI ($/hour)
- DBH revenue / hours spent = $/hour
- Target: High-ROI hours ($200+/hour) vs. low-ROI ($50-100/hour)

### Learning ROI (Knowledge value)
- PG learning: Curriculum day progression / hours
- Skill development: New capability per hour
- Score: 1-10 based on value to future goals

### Well-being ROI (Sanity/relationships)
- Sleep quality/quantity: Essential foundation
- Relationship investment: Needed for long-term support
- Personal development: Skills/character that age well
- Score: 1-10 based on contribution to sustainable success

## Time Allocation Baseline
- Total weekly: 20 hours (Tom's commitment)
- Current split: DBH ~12h, PG ~4h, Personal ~4h
- Can vary week-to-week based on priorities

## Warning Signs
- **Burnout risk (RED):** If sleep <6hrs/night for 2+ weeks + focus declining
- **Efficiency declining:** If $/hour trending down 3+ weeks
- **Imbalance:** If >70% of time in one domain, others declining
- **Creep:** If total time consistently >20h/week

## Reallocation Strategy
- Short-term (1-2 weeks): Tactical shifts for immediate goals
- Medium-term (1 month): Sustainable reallocation based on priorities
- Long-term (3+ months): Structural changes to how time is allocated

## ROI Calculation Formula
```
Simple ROI: (Value - Cost) / Cost
For DBH: ($Revenue - $Cost of hours) / $Cost of hours
For Learning: Learning units gained / hours invested
For Well-being: Sustainability score / hours invested
```

## Burnout Prevention
If personal dev time drops below 4h/week for 3+ weeks, flag for burn risk.
If sleep or focus metrics decline, prioritize recovery in next week.
Some low-ROI time is essential for long-term sustainability.
