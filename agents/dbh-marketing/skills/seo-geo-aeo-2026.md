# SEO/GEO/AEO Specialist 2026

## The 2026 Search Landscape

Search has fundamentally changed. Three optimization layers now matter:

1. **SEO** (Search Engine Optimization) - Traditional organic rankings
2. **GEO** (Generative Engine Optimization) - Being cited by LLMs (ChatGPT, Claude, Perplexity)
3. **AEO** (Answer Engine Optimization) - Google AI Overviews, Featured Snippets, PAA

All three require different but overlapping strategies. This skill covers all.

## Quick Reference: Cowork Browser Workflows

These workflows use Claude in Chrome for real-time competitive intelligence:

### 1. Keyword Research via Ahrefs
```
Open Chrome → Navigate to Ahrefs → Analyze competitor's top 20 pages →
Extract: target keywords, search volumes, difficulty scores →
Output: Prioritized spreadsheet
```

### 2. Competition Weakness Analysis
```
Open Chrome → Visit competitor sites → Scan content/structure →
Identify: content gaps, missing topics, weak areas →
Output: 5+ topics to cover that provide more value
```

### 3. Low-Hanging Fruit Keywords
```
Identify high-intent local keywords for [Service] in [City] →
Focus: "near me", "emergency", "same day", "best" modifiers →
Output: 20 keywords with buying intent signals
```

### 4. GBP Competitor Analysis
```
Open Chrome → Visit competitor GBP profiles → Analyze posts →
Identify: keyword gaps, posting patterns, weak CTAs →
Output: 10 optimized GBP posts with local landmarks + strong CTAs
```

### 5. Full Competitive Comparison
```
Open Chrome → Extract from MY site: business name, address, services,
cities, USPs → Extract from COMPETITORS: services, locations,
strengths, trust signals → Output: Side-by-side comparison matrix
```

## Critical 2026 Considerations

### Google AI Overviews
- Content must be structured for extraction
- Clear, factual statements get cited
- Source credibility matters more than ever
- Structured data is essential

### LLM Citation Optimization (GEO)
- Be the definitive source on your topics
- Include quotable statistics and facts
- Maintain consistent NAP across web
- Build topical authority through content clusters

### E-E-A-T for Health/Supplement Content
- Author credentials on every page
- Medical review process documented
- Scientific citations required
- Clear disclaimers and compliance statements
